# 🚀 Getting Started

Welcome to the developer guide for Tymli.
This section will help you set up the project and run it locally across different platforms.

---

## Requirements

Before jumping in, make sure you have:

- [Node.js](https://nodejs.org/) (v18 or newer)
- [pnpm](https://pnpm.io/) (recommended)
- [Git](https://git-scm.com/)
- PostgreSQL & Redis (for backend)
- Android Studio / Xcode *(for mobile)*
- VS Code (recommended)

---

## Project Layout

This is a single-repo project. You can build and test:

- 🖥️ Desktop app (Electron)
- 📱 Mobile app (React Native)
- 🔧 Backend API (Node.js + PostgreSQL)
